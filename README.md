# TXT 阅读器

一个安静、清晰、面向中文阅读的 macOS 桌面 TXT 阅读器，支持各种文本格式。基于 Electron + React + Vite 构建。

## 下载

从 [GitHub Releases](https://github.com/MartyYao/txt-reader/releases/latest) 下载最新版本。

当前发布包：
- `TXT.Reader-1.0.0-arm64.dmg`：macOS Apple Silicon 安装包
- `TXT.Reader-1.0.0-arm64-mac.zip`：macOS Apple Silicon 压缩包

说明：当前构建未使用 Apple Developer ID 签名。首次打开时，macOS 可能提示“无法验证开发者”；可在“系统设置 → 隐私与安全性”中允许打开。

## 已实现功能

### 文件操作
- **打开本地 TXT 文件**：按钮选择（系统文件对话框）和拖拽打开
- **最近打开记录**：自动记录最近 50 个文件，侧边栏可快速访问
- **阅读位置恢复**：再次打开同一文件时恢复到上次阅读位置

### 编码支持
- **自动检测**：使用 jschardet 自动识别文件编码，优先适配中文场景
- **手动切换**：设置面板中可手动选择编码（UTF-8、GBK/GB18030、Big5、Shift JIS 等）
- **常见编码映射**：自动将误检测的 ISO-8859-1/GB2312 等映射到正确编码

### 阅读界面
- **标题/文件信息**：工具栏显示文件名，阅读区顶部显示文件信息
- **滚动进度**：底部状态栏和阅读区信息栏实时显示阅读百分比
- **阅读宽度可调**：400–1200px 范围内自定义阅读宽度

### 中文长文本体验
- **字体大小**：12–40px 可调（快捷键 `⌘+` / `⌘-`）
- **行高**：1.2–3.0 可调
- **三种主题**：浅色 / 护眼（暖黄） / 深色（快捷键 `⌘T`）
- **设置持久化**：所有设置自动保存到本地

### 目录/章节侧栏
- **自动识别章节**：支持常见中文小说章节格式（"第X章"、"第X节"、"卷X"、"序章/终章/楔子/引子/后记/番外"等）
- **章节跳转**：点击侧边栏目录项快速跳转
- **当前章节高亮**：自动标注当前正在阅读的章节

### 搜索
- **全文搜索**（快捷键 `⌘F`）
- **上一个/下一个结果**：键盘 ↑↓ 或 `Shift+Enter` / `Enter`
- **匹配高亮**：所有匹配项高亮，当前项特殊颜色标记

### 书签
- **添加书签**（快捷键 `⌘B`）：在当前阅读位置添加书签，自动抓取附近文本作标签
- **删除书签**：侧边栏书签列表中一键删除
- **书签跳转**：点击书签快速跳转
- **持久保存**：书签按文件路径保存到本地

### 朗读
- **系统语音合成**：使用 macOS 内置语音引擎
- **开始/暂停/恢复/停止**：工具栏播放控制按钮
- **自动分段**：长文本自动分句朗读
- **不可用时禁用**：Speech Synthesis 不可用时按钮自动变为禁用状态

### 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| `⌘O` | 打开文件 |
| `⌘F` | 搜索 |
| `⌘B` | 添加书签 |
| `⌘+` / `⌘-` | 增大/减小字号 |
| `⌘T` | 切换主题 |
| `⌘S` | 切换侧边栏 |
| `Esc` | 关闭搜索/设置面板 |

## 安装与运行

### 环境要求
- Node.js >= 18
- npm >= 9
- macOS 10.15+（Windows/Linux 也可运行但未经充分测试）

### 安装

```bash
cd "/Users/mengchao/Claude code/TXT阅读器"
npm install
```

### 开发模式运行

```bash
npm run dev
```

这会同时启动 Vite 开发服务器（端口 5173）和 Electron 窗口。

### 构建生产版本

```bash
npm run build
```

构建产物在 `release/` 目录下（macOS 下为 `.dmg` 和 `.zip`）。

### 仅构建渲染层（验证）

```bash
npm run build:renderer
```

构建产物在 `dist/` 目录下。

## 示例文件

`sample/示例小说.txt` 包含一个带有多级章节标题（序章、第一章、第二章、第三章）的中文小说片段，可用于测试章节检测、书签、搜索等功能。

## 限制与后续计划

### 当前限制
- **仅支持 TXT 格式**：不支持 EPUB、MOBI、PDF 等格式读取
- **无导出功能**：不支持导出为图片或其他格式
- **无字体选择**：使用系统默认字体，不支持自定义字体上传
- **大文件性能**：超过 50MB 的 TXT 文件可能出现加载延迟
- **无标签/分组**：书签和最近文件不支持分组或标签
- **无云同步**：所有数据存储在本地，不支持跨设备同步
- **章节检测有限**：仅支持常见中文/英文章节格式，生僻格式可能漏检
- **搜索无正则**：仅支持纯文本搜索，不支持正则表达式
- **朗读语言限制**：优先使用系统 zh-CN 语音，其他语言需系统安装对应语音包

### 后续计划
- EPUB/MOBI/PDF 格式支持
- 正则搜索
- 自定义字体与更多排版选项
- 笔记与高亮批注
- 多标签页阅读
- 垂直书写（竖排）模式
- 更多主题与自定义主题
